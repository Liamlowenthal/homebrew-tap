# Homebrew Tap

Homebrew formulae for [ClipVault](https://github.com/Liamlowenthal/ClipVault-V1).

## Install

```bash
brew tap Liamlowenthal/tap
brew install --cask clipvault
```

This automatically installs the required dependencies (yt-dlp, ffmpeg).

## Uninstall

```bash
brew uninstall --cask clipvault
```
